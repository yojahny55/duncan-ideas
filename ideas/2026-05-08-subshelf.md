# SubShelf — The Subscription Graveyard Manager

**Date:** 2026-05-08
**Author:** Duncan ⚔️ (Daily App Idea)
**Inspired by:** Subscription fatigue trending across X/Twitter — users drowning in recurring payments they forgot about, can't cancel, or don't even use.

---

## Problem Statement

The average person now manages **12+ recurring subscriptions** (streaming, SaaS, apps, gaming, fitness). The result:

- **$300+/year wasted** on forgotten or unused subscriptions
- **Cognitive overload** — billing cycles scattered across dates, payment methods, and emails
- **Dark patterns** — companies make cancellation intentionally hard
- **No unified view** — bank statements don't tell you *what* a charge is for
- **Guilt/spending anxiety** — "Am I really using all of this?"

Current solutions (Rocket Money, Truebill) require bank access, sell your data, or charge *another subscription* to cancel subscriptions. Ironic.

## Solution

**SubShelf** — a lightweight, privacy-first subscription tracker that lives in your browser (and eventually your phone). No bank linking required.

### Core Concept
A visual "bookshelf" metaphor where each subscription is a book. You see everything at a glance — what's active, what's gathering dust, what's costing you the most.

### Key Features
1. **Visual Bookshelf UI** — Drag-and-drop subscriptions onto shelves (Active / Paused / Cancelled)
2. **Manual Entry, Zero Data Harvesting** — You add what you want. No bank scraping, no selling anonymized data
3. **Cost Forecasting** — "This month you'll spend $187. Next month $220 (annual renewal for Spotify)"
4. **Usage Check-In** — Weekly nudge: "Did you use Hulu this week?" After 3 "no" answers, it suggests cancelling
5. **Cancellation Cheat Sheet** — Community-sourced guides on *how* to cancel each service (click path, phone number, direct link)
6. **Subscription Sharing** — See which friends use the same service; split family plans
7. **Import from Email** — Forward subscription receipts to a SubShelf address; auto-parsed and added

## Tech Stack

| Layer | Choice | Why |
|-------|--------|-----|
| **Frontend** | React + TypeScript + Tailwind | Fast, responsive, works everywhere |
| **State** | Zustand | Lightweight, no boilerplate |
| **Backend** | Supabase (PostgreSQL + Auth + Edge Functions) | Zero server management, generous free tier |
| **Email Parsing** | Cloudflare Email Workers | Receive forwarded receipts, parse with LLM |
| **Mobile** | React Native (Expo) | Share codebase with web |
| **Hosting** | Vercel (web) + Expo EAS (mobile) | Free tier friendly |
| **AI Features** | OpenAI API (optional) | Smart categorization, cancellation summaries |
| **Payments** | Stripe (for premium tier) | Standard |

## Monetization

| Tier | Price | Features |
|------|-------|----------|
| **Free** | $0 | Up to 15 subscriptions, basic dashboard, manual entry |
| **Pro** | $2.99/mo or $24/yr | Unlimited subs, email import, usage check-ins, cancellation guides, sharing |
| **Business** | $9.99/mo | Team subscription tracking, expense reports, tax export |

**Anti-irony rule:** We will *never* charge a subscription to cancel your subscriptions. Pro features are about *insight*, not basic functionality.

**Future revenue streams:**
- Affiliate deals for better alternatives ("Switch to annual and save $30" — we get a cut)
- Anonymous, aggregated subscription trend data (opt-in only)
- API for financial advisors / bookkeepers
