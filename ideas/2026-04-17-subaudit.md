# SubAudit — Know What You're Paying For

**Date:** 2026-04-17
**Category:** Personal Finance / SaaS
**Status:** Idea ✨

---

## Problem Statement

The average person has **12+ active subscriptions** (streaming, SaaS, apps, memberships) and wastes **$300+/year** on forgotten or redundant ones. There's no unified view — subscriptions are scattered across credit cards, Apple/Google Play, PayPal, and bank statements. People don't realize they're paying for overlapping services (Spotify + Apple Music + YouTube Premium) or auto-renewed trials they never used.

**Key pain points:**
- No single dashboard for all subscriptions across payment methods
- Forgotten free trials converting to paid
- Overlapping/redundant services
- Price hikes go unnoticed
- Hard to compare "bundle" alternatives

## Solution

**SubAudit** — a subscription intelligence app that:

1. **Auto-discovers** subscriptions by scanning email receipts (Gmail/Outlook OAuth), bank/credit card transactions (Plaid API), and Apple/Google Play purchase history
2. **Classifies** each subscription by category (streaming, productivity, fitness, etc.)
3. **Detects waste:** duplicate services, unused trials, price increases, better alternatives
4. **Monthly audit report** with one-tap cancel actions or "switch to bundle" suggestions
5. **Price tracking** — alerts when a subscription raises its price
6. **Shared plan manager** — track who owes what on family/team plans

## Target Users

- **Primary:** 25-45 year olds with 5+ subscriptions ($500+/year spend)
- **Secondary:** Small business owners managing team SaaS subscriptions
- **Tertiary:** Parents managing family plans (Spotify Family, YouTube Premium, etc.)

## Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | React Native (Expo) — iOS + Android + Web |
| Backend | Node.js + Fastify on Railway/Fly.io |
| Email Parsing | Google/Gmail API + Microsoft Graph API |
| Bank Linking | Plaid API |
| Database | PostgreSQL + Redis (caching) |
| Auth | Clerk (or Supabase Auth) |
| Notifications | OneSignal (push) + Resend (email reports) |
| AI/NLP | OpenAI API for subscription classification from receipt text |
| Infra | Docker, GitHub Actions CI/CD |

## Monetization

| Tier | Price | Features |
|------|-------|----------|
| Free | $0 | Up to 5 subscriptions tracked, basic alerts |
| Pro | $4.99/mo | Unlimited subscriptions, email scanning, monthly audit, cancel assist |
| Family | $7.99/mo | Pro + shared plan tracking, up to 6 members |
| Business | $14.99/mo | Team SaaS management, expense export, admin dashboard |

**Affiliate revenue:** Commission from "switch to this plan" recommendations (negotiated deals with streaming/SaaS companies).

## Competitive Landscape

- **Truebill/Rocket Money** — exists but $6-12/mo, heavy upsell, privacy concerns
- **Subtrack** — basic, no bank linking
- **Differentiation:** Cheaper, email-first discovery (no bank required for basic tier), open-source core, family plan focus

## MVP Scope (v1)

- Email scanning (Gmail) for subscription receipts
- Manual add/edit subscriptions
- Dashboard with spend breakdown
- Price increase alerts
- Monthly summary email
