## Problem Statement

Context switching is devastating for productivity. When you're deep in a task and get interrupted (meeting, urgent Slack, different project), you lose:

1. **Mental context** ‚Äî What were you thinking? What was your next step?
2. **Digital context** ‚Äî Which tabs, files, and apps were open?
3. **Momentum** ‚Äî The 23+ minutes it takes to get back into flow state

Current solutions (browser tab managers, workspace apps) save the *what* but miss the *why* ‚Äî the mental notes about where you were and what you were doing.

## Proposed Solution

**FlowState** is a work session preservation system that:

1. **Captures context instantly** ‚Äî One hotkey when interrupted
2. **Records mental state** ‚Äî Voice note or quick text about what you were doing
3. **Snapshots digital state** ‚Äî Open tabs, files, apps, clipboard
4. **Restores everything** ‚Äî One click to reopen exactly where you were

Think of it as "hibernate mode" for your brain + computer combined.

## Key Features

### 1. Quick Capture (Cmd+Shift+S)
- One hotkey to save current session
- Optional voice note: "I was debugging the auth flow, next step is checking the token expiry"
- Auto-captures: browser tabs, open files, active app windows

### 2. Session Timeline
- Visual timeline of saved sessions
- Search by project, date, or content
- Preview what was open before restoring

### 3. Smart Restoration
- Reopens tabs and files exactly as they were
- Shows your voice note/text reminder
- Optional: restore clipboard contents

### 4. Context Tags
- Auto-tag by project (based on folder/repo)
- Manual tags for organization
- Filter and group sessions

### 5. Interruption Log
- Track what interrupted you (meeting, Slack, phone)
- Patterns: "You get interrupted most at 2-3pm"
- Insights for protecting focus time

## Technical Approach

### MVP (Browser Extension + Desktop App)
- **Browser extension:** Capture/restore tabs, inject capture UI
- **Desktop companion:** Window positions, open files (using accessibility APIs)
- **Local storage:** Privacy-first, all data stays local
- **Optional sync:** Encrypted cloud backup

### Stack
- **Extension:** Chrome/Firefox/Edge (WebExtension API)
- **Desktop:** Electron or Tauri (cross-platform)
- **Storage:** Local SQLite + optional encrypted cloud
- **Voice:** Web Speech API for transcription
